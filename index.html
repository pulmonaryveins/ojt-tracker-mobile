  • At least 100 MB free storage space
  • Camera permission for photo documentation
  • Storage permission for saving reports
  • Notification permission for reminders

Installation Guide

1
Enable Unknown Sources Go to Settings → Security → Enable "Install from Unknown Sources" or "Allow from this source"
2
Download APK Click the download button above to get the latest OJT-Tracker-v1.0.0-release.apk file
3
Install App Open the downloaded APK file and follow the installation prompts
4
Grant Permissions Allow the app to access Camera, Storage, and Notifications when prompted
5
Start Tracking Open the app, create your profile, and start tracking your OJT activities!
